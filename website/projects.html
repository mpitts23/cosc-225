Titanic Kaggle Competition

Last year, I participated in a data science Kaggle competition as part of my Machine Learning class. I used tools such as random forests and support vector machines (SVMs) to predict which passengers survived on the Titanic. Many variables, like age, social class, and gender, were very significant predictors of survival.

Using ARIMA and GARCH to model financial time series data

This is a project I completed as part of the capstone STAT-495 class. It has helped me learn common ways to model time series data.

ARIMA stands for auto-regressive indifference moving average.

GARCH stands for generalized auto-regressive conditional heteroskedasticity.
